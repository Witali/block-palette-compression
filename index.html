Block Palette Compression

BPAL compression, viewing, and shader texture demos. Read the description.

Codec laboratory Test BPAL, DCTBS2, and BPDH in one workspace with a shared comparison view and block inspector. Block palette compression Compress blocks with local color indices that reference one shared image palette. DCT compression Compress into fixed 16×16 DCT records and read any reconstructed pixel directly by coordinate. Hybrid BPAL/DCT compression Select BPAL or DCT independently for every 16×16 unit under one payload target. Image Viewer Open BPAL, BPLM, BPDH, or regular images and inspect them with zoom and pan controls. Cube Render BPAL or DCTBS2 directly in the fragment shader. BPAL texture sampler Sample BPAL mip levels in a shader with bilinear, trilinear, or anisotropic filtering.
